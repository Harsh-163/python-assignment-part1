# Student Grade Tracker — Python Assignment (Part 1)

This repository contains the implementation of a **Student Grade Tracker** using core Python concepts. The project focuses on data cleaning, processing, and reporting for academic datasets.

---

## 📋 Project Modules

### 1. Data Parsing & Profile Cleaning
Transforming unstructured raw data into clean, usable formats.
* **Concepts:** `.strip()`, `.title()`, `int()` casting, and List Comprehension.
* **Validation:** Name verification using `.isalpha()`.

### 2. Marks Analysis & Dynamic Entry
Automated grading and a real-time system to add new data.
* **Grading Logic:** A+ (90-100) down to F (<60).
* **Statistics:** Automatic calculation of Total, Average, Max, and Min scores.
* **Error Handling:** Input validation to prevent crashes from non-numeric or out-of-range marks.

### 3. Class Performance Summary
Generating a structured report for multiple students.
* **Data Structure:** Handling Lists of Tuples.
* **Formatting:** Tabular output using f-string alignment (`:<18`).
* **Aggregate Data:** Identifying the Class Topper and overall Class Average.

### 4. String Manipulation Utility
Advanced text processing of student essays.
* **Operations:** Case-insensitive counting, substring replacement with Emojis, and sentence-level tokenization.
* **Automation:** Automatic punctuation check and numbered list generation.
